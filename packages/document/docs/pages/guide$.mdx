# 指南

## 快速开始
本节将介绍如何在项目中使用 wind-design

### 用法

#### 引入 wind-design

```tsx
import { Button } from 'wind-design';
import 'wind-design/dist/style.css';

const App = () => {
  <>
    <Button btnType="primary">Button</Button>
  </>
}

export default App
```

好了，现在你能够在页面上看待蓝色的按钮组件了，接下来开发中需要什么组件的话就引入什么组件。
